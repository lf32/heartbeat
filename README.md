# heartbeat

This is a project focused around Capture The Flag based on flask.

### Features
* [x] Reverse Shell Generator
* [x] Cryptography Cracker
* [x] Stegnographic Analyzer

and more...

### Local Development

```bash
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
flask run
```
